# Lístkomat v2.0 — Brno Live Map Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a full-screen, portrait live map of Brno's trams/trolleybuses/buses + stops, reached via a "Živá mapa" button on the Brno ticket screen.

**Architecture:** A source-agnostic data layer (`VehicleSource` protocol → `BrnoVehicleSource` decoding the keyless KORDIS ArcGIS GeoJSON; Prague slots in later). Static stops are bundled (`Resources/brno-stops.json`, generated by a script). The map is `MKMapView` wrapped in `UIViewRepresentable` (keeps the iOS 16.2 floor); a `LiveMapViewModel` polls every ~8 s and animates moves. Entry is gated by a `hasLiveMap` flag on `City`.

**Tech Stack:** Swift, SwiftUI, MapKit (`MKMapView` via `UIViewRepresentable`), `URLSession`+`Codable`. No new dependencies. Tests: XCTest with inline-JSON fixtures (matches `ListkomatTests/CatalogTests.swift`).

**Step 3: Implement the decoder.** In `BrnoVehicleSource.swift`:
```swift
import Foundation
import CoreLocation

enum BrnoVehicleSource {
    private struct FeatureCollection: Decodable { let features: [Feature] }
    private struct Feature: Decodable { let geometry: Geo; let properties: Props }
    private struct Geo: Decodable { let coordinates: [Double] }   // [lng, lat]
    private struct Props: Decodable {
        let LineName: String; let Bearing: Double; let TimeUpdated: Double
        let IsInactive: String; let VType: Int; let ID: Int
    }

    static func decode(_ data: Data) throws -> [Vehicle] {
        let fc = try JSONDecoder().decode(FeatureCollection.self, from: data)
        return fc.features.compactMap { f -> Vehicle? in
            guard f.properties.IsInactive != "true",
                  f.geometry.coordinates.count == 2 else { return nil }
            return Vehicle(
                id: String(f.properties.ID),
                coordinate: CLLocationCoordinate2D(latitude: f.geometry.coordinates[1],
                                                   longitude: f.geometry.coordinates[0]),
                bearing: f.properties.Bearing >= 0 ? f.properties.Bearing : nil,
                line: f.properties.LineName,
                kind: kind(forVType: f.properties.VType),
                updatedAt: Date(timeIntervalSince1970: f.properties.TimeUpdated / 1000))
        }
    }

    // VType codes to be confirmed by sampling known lines (Step 5). Default bus.
    static func kind(forVType v: Int) -> VehicleKind {
        switch v { case 1: return .tram; case 2: return .trolleybus; default: return .bus }
    }
}
```

**Step 4:** Run tests. Expected: PASS (both).

**Step 3: Implement** the helper + the query URL builder:
```swift
extension BrnoVehicleSource {
    static func layerName(year: Int) -> String { "Kordis_\(year % 100)_polohy" }

    static func currentQueryURL(now: Date = Date(),
                                calendar: Calendar = .init(identifier: .gregorian)) -> URL {
        let year = calendar.component(.year, from: now)
        let base = "https://gis.brno.cz/ags1/rest/services/Hosted/\(layerName(year: year))/FeatureServer/0/query"
        return URL(string: "\(base)?where=1%3D1&outFields=*&f=geojson")!
    }
}
```
(Year-based is the primary path; a portal-item fallback can be added later if a New-Year miss is ever observed. Keep YAGNI for v2.0.)

**Step 4:** Run → PASS.

**Step 5: Add the network fetch + conform to a protocol.** Create `Listkomat/Services/VehicleSource.swift`:
```swift
protocol VehicleSource { func fetch() async throws -> [Vehicle] }
```
Make `BrnoVehicleSource` a `struct` conforming to it (keep the static `decode`/`kind`/`layerName` for tests):
```swift
struct BrnoLiveSource: VehicleSource {
    var session: URLSession = .shared
    func fetch() async throws -> [Vehicle] {
        let (data, _) = try await session.data(from: BrnoVehicleSource.currentQueryURL())
        return try BrnoVehicleSource.decode(data)
    }
}
```

enum StopsStore {
    static func decode(_ data: Data) throws -> [Stop] { try JSONDecoder().decode([Stop].self, from: data) }

        guard let url = Bundle.main.url(forResource: "brno-stops", withExtension: "json"),

        return (try? decode(data)) ?? []
    }
}
